# QA Visual Checklist

## Primeira impressao

- A pagina parece premium nos primeiros cinco segundos?
- A proposta de valor esta clara sem scroll?
- A marca parece confiavel para uma PME?
- O design parece proprio, nao template generico?

## Layout

- Existe alinhamento consistente?
- As margens sao coerentes?
- As secoes têm ritmo visual?
- Nao ha elementos colados ou perdidos?
- O conteudo respira sem parecer vazio?

## Tipografia

- H1 tem impacto e legibilidade?
- H2 e H3 criam hierarquia clara?
- O corpo de texto e facil de ler?
- Labels, badges e notas sao consistentes?
- Nao ha linhas demasiado longas?

## Cores

- O contraste e suficiente?
- A paleta transmite confianca?
- Os acentos sao usados com moderacao?
- Nao existe aspeto barato, neon ou generico?

## Componentes

- Botoes têm estados de hover e focus?
- Cards têm funcao clara?
- Formularios sao curtos e legiveis?
- Tabelas de pricing sao faceis de comparar?
- CTAs usam texto especifico?

## Conteudo

- A copy fala de resultados, nao apenas funcionalidades?
- Os pacotes estao claros?
- Os planos mensais estao claros?
- A diferenca entre projeto e mensalidade esta explicada?
- Existe caminho natural para contacto?

## Mobile

- A primeira dobra continua forte?
- Botoes sao faceis de tocar?
- Tabelas continuam legiveis?
- Cards nao ficam demasiado longos?
- Nao ha overflow horizontal?

## Performance visual

- Nao ha layout shift perceptivel?
- Imagens e elementos visuais carregam bem?
- Animacoes sao suaves?
- Nada importante depende de animacao para aparecer?

## Regra final de aprovacao

So aprovar se a experiencia parecer digna de uma empresa premium de websites, automacao, CRM e IA para PMEs.
